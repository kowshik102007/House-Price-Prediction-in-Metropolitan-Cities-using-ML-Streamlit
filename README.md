# 🏠 House Price Prediction in Metropolitan Cities using ML

This project predicts house prices in major metropolitan cities using Machine Learning.  
It provides an interactive web application built with Streamlit where users can enter property details and get an estimated price instantly.

---

## 🚀 Features

- Predict house prices based on user input  
- Supports multiple metropolitan cities  
- Simple and interactive Streamlit interface  
- Real-time prediction  

---

## 🧠 Technologies Used

- Python  
- Machine Learning (scikit-learn)  
- Streamlit  
- Pandas & NumPy  

---

## ⚙️ How to Run

1. Clone the repository:

```
git clone https://github.com/kowshik102007/House-Price-Prediction-in-Metropolitan-Cities-using-ML-Streamlit.git
cd House-Price-Prediction-in-Metropolitan-Cities-using-ML-Streamlit
```

2. Install required packages:

```
pip install -r requirements.txt
```

3. Run the Streamlit app:

```
streamlit run Code.py
```

4. Open the local URL shown in the terminal.
